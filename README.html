This application is built to retrieve data from http://spdrs.com and convert it to CSV.
It can be executed by entering etfsearch.ddns.net/phpfiles/Assignment/Login.html as the url in any browser.
The CSV file is automatically created once the user searches for an ETF symbol.
The CSV file is automatically downloaded and placed into the Assignments directory, located at etfsearch.ddns.net/phpfiles/Assignment/
Users are able to register new accounts as well as log in to already existing ones
